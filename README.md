# CareerPilot AI

CareerPilot AI is an AI-powered career success platform that helps job seekers discover opportunities, analyze skill gaps, track applications, manage interviews, and improve employability through personalized recommendations.

## Features

* User Authentication
* Resume Upload & Analysis
* Skill Gap Detection
* AI Job Recommendations
* Job Application Tracking
* Interview Scheduling
* Recruiter Dashboard
* Notification System
* Analytics Dashboard

## Tech Stack

### Frontend

* React
* Material UI
* Axios

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* JPA/Hibernate

### Database

* MySQL

### Deployment
